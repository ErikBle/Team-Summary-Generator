# Team-Summary-Generator

## General Info:
A node.js app that generates team summaries.

## Programs/Languages/Libraries used:
* Html
* CSS
* Javascript
* JQuery
* Node.js
* Inquirer

## Instructions:
Use "npm install" to get the proper packages.
<br>
Then run "node index.js" to start the app.
